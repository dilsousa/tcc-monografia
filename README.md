## Como gerar PDF com docker local
* Clean: `rm -rf latex-docker && rm monografia.*`
* Clone: `git clone https://github.com/andreformento/latex`
* Generate: `latex/dockercmd.sh /bin/sh -c "pdflatex tex/monografia.tex"`
